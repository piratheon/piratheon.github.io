## Flagship — `rika-agent`

> *An assistant that lives on your machine, not someone else's cloud.*

[![Stars](https://img.shields.io/github/stars/piratheon/rika-agent?style=flat-square&color=8fe2bb&labelColor=0e1311)](https://github.com/piratheon/rika-agent/stargazers)
[![License](https://img.shields.io/github/license/piratheon/rika-agent?style=flat-square&color=4d564f&labelColor=0e1311)](https://github.com/piratheon/rika-agent/blob/main/LICENSE)
[![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white&labelColor=0e1311)](https://github.com/piratheon/rika-agent)

A persistent, self-hosted agentic AI system that integrates with your shell, file system, and network. Designed for developers who choose extensibility over managed convenience.

```
Function calling  ·  AES-256-GCM key storage  ·  3-tier sandboxing
Background monitoring  ·  Telegram interface  ·  Ollama / G4F inference
```

→ **[View source](https://github.com/piratheon/rika-agent)** · **[Quick-start](https://github.com/piratheon/rika-agent#quick-start)**

## Principles

**Linux-first** — Unix environments, POSIX standards, terminal workflows. Not a cross-platform compromise.

**Security-aware** — Encryption and sandboxing are first-class design constraints, not afterthoughts.

**Control over convenience** — Every action a tool takes should be inspectable and reversible.

**System-oriented** — Process management, I/O, memory, networking. Abstractions are a ceiling, not a floor.
